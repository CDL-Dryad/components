# Installation

<Meta title="Documentation/Installation" />

The Frictionless Components project can be installed via NPM or CDN.

## NPM

Install the library as you usually do with NPM:

```bash
$ npm install frictionless-components
```

Then you can import components you want with its styles:

```javascript
import 'frictionless-components/lib/styles'
import frictionlessComponents from 'frictionless-components'
```

## CDN

Alternatively, it might be linked directly from the CDN ([available distributions](https://unpkg.com/browse/frictionless-components/)):

```html
<link rel="stylesheet" href="//unpkg.com/frictionless-components/dist/frictionless-components.css">
<script src="//unpkg.com/frictionless-components/dist/frictionless-components.js"></script>
```

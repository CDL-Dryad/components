# Usage

<Meta title="Documentation/Usage" />

The package provides a set of components with their styles in `lib/styles` directory and a render function to render it to the DOM:

```javascript
import 'frictionless-components/lib/styles'
import frictionlessComponents from 'frictionless-components'

const element = document.getElementById('app')
frictionlessComponents.render(frictionlessComponents.Report, {report}, element)
```

You can import only a component you need, for example, a Report:

```javascript
import 'frictionless-components/lib/styles/report.css'
import { Report } from 'frictionless-components/lib/components/Report'
import { render } from 'fictionless-components/render'

const element = document.getElementById('app')
render(Report, {report}, element)
```
